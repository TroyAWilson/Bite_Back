# Bite Back

Bite Back is a small top-down action game made for the Brainless Game Jam:
https://itch.io/jam/brainless-game-jam

You play as a tiny fighter, avoid enemy hits, and strike back with quick melee attacks.

## Built With

- Godot 4.3

## How To Run

1. Download or clone this repository.
2. Open Godot and import the project folder.
3. Press Play to run the main scene.

## Controls

- Move: Arrow keys (or any keys mapped to Godot `ui_*` movement actions)
- Attack: `X`

## Project Structure

```text
autoload/   Global singleton scripts (audio controller)
scenes/     Main game scenes and scene scripts
scripts/    Core gameplay scripts (player, camera, hit/hurt boxes)
audio/      Music and sound effect imports
sprites/    Sprite sheets and texture assets
```
