# ![ant down](/ant-down.gif) Bite Back ![ant down](/ant-down.gif)

Bite Back is a small top-down action game made for the Brainless Game Jam:

[![Brainless Game Jam](/brainless.png)](https://itch.io/jam/brainless-game-jam)


You play as a little ant guy, avoid the big hand, and strike back with quick melee attacks.

## Built With

- Godot 4.3

## How To Run

1. Download or clone this repository.
2. Open Godot and import the project folder.
3. Press Play to run the main scene.

## Controls

- Move: Arrow keys (or any keys mapped to Godot `ui_*` movement actions)
- Attack: `X`
